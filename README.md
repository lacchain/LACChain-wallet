# Lacchain Wallet

## Requirements

- node Version: **14.17.0**

## Environment Variables customization

Create a copy of env.example:

```sh
cp env.example .env
```

And then you can customize `.env` file accordingly.

## Building

Make sure you have `.env` file properly set.

```sh
yarn start
```

## Running

Make sure you have `.env` file properly set.

```sh
yarn build
```
